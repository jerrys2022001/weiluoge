Usage Tips

  • When you see a 16-bit UUID in logs (for example 0x180F or 0x2A37), locate it in the Assigned Number column first.
  • Use the UTI column to map entries to canonical Bluetooth naming (org.bluetooth.*).
  • Use the Type column to quickly separate Service, Characteristic, Descriptor, and Attribute UUIDs.
  • For automation and indexing, consume the CSV export instead of scraping HTML.
