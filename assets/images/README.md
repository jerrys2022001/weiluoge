# Image Source Policy

Preferred image sources for this repository:

1. `https://unsplash.com`
2. `https://www.pexels.com`
3. `https://pixabay.com`

Rules:

- Reuse images that are already in active site pages when possible.
- When adding new stock images, choose technology scenes or people/workflow scenes first.
- Avoid adding general-purpose stock pools from non-preferred sources unless explicitly requested.
- Record the source site and original URL in the folder README for every new curated batch.
